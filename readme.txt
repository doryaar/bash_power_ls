power-ls is a tool for scanning files from target directory and into depth
this version echo all files and dir names include ident of depth for dir inside dir

you can use options:
-ru means start at cd /, or root folder
-d means target spesific directory

Arguments at command line input in the format of:

	-option 
	or
	-option parameter

Examples: For target root folder use ./script -ru
	or
	For target spesific folder use ./script -d /home/Desktop
